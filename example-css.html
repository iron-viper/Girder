Girder

Example Using Just CSS
Home Work Blog Contact

A minimalistic grid for building awesome Websites with Sass targeting the modern browser. It uses silent classes (placeholders) to structure content in HTML and keeps your Markup free of excess presentational classes like "unit_1of4", "small-2", "grid4". This page is built with Girder.

Installation

Copy the Girder files from Github or from the downloaded examples directly into your project's path.

With Bower just use: "bower install girder --save" then add a reference to either scss/_girder.scss or girder.css inside the components folder.

Using Scss

@import the _girder.scss partial which should reside next to its modules in your project.

Standard CSS

Simply include the girder.css file in your document before your site's custom styles.
